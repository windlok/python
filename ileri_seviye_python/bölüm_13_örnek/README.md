# bölüm_13_örnek — Chat örneği (2 farklı uygulama)

Bu klasörde aynı “chat” fikrinin iki farklı sürümü var:

- [benim_yaptıgım](benim_yaptıgım) — kişisel deneme
- [hocanın_yaptigi](hocanın_yaptigi) — daha temiz ping-pong akışı

Her iki sürüm de `socket` ile **tek client - tek server** mantığında çalışır (çoklu kullanıcı yok).

## Hızlı çalışma

- Önce seçtiğin sürümün server'ını bir terminalde çalıştır.
- Sonra aynı sürümün client'ını ikinci terminalde çalıştır.
- Bağlantıyı bitirmek için ilgili komutu kullan:
	- kişisel sürüm: `exit`
	- hoca sürümü: `quit`

Detaylar için alt klasörlerin README'lerine bak.
